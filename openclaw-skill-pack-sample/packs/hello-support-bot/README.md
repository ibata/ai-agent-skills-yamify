# hello-support-bot

Minimal OpenClaw skill pack for Yamify testing.

## What this pack contains
- `openclaw-pack.yaml` (required manifest)
- `prompts/system.txt` (default assistant behavior)
- `skills/triage.md` (example support triage playbook)
- `skills/escalation.md` (example escalation rules)

## Yamify Skill Pack modal values
- **Pack Repository**: `https://github.com/<org>/<repo>`
- **Pack Git Ref**: `main` (or your branch)
- **Pack Path**: `packs/hello-support-bot`
